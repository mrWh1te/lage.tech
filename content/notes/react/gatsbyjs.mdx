Helpful tid-bits to blog management with GatsbyJS

## Basic File Management without Bells & Whistles

Use the project root directory's `static` folder to include any additional assets during build for copying into the `public` directory of the build.

- [GatsbyJS Docs: Static Folder](https://www.gatsbyjs.com/docs/how-to/images-and-media/static-folder/)
